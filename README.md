# Swam One-Page Portfolio (GitHub Pages)

## Structure
- index.html
- assets/
  - css/style.css
  - js/main.js
  - img/ (put your images here)
  - resume.pdf (replace with your real resume)

## Edit these placeholders
- Replace `YOUR_EMAIL@example.com` in `index.html`
- Replace `DEST_EMAIL` in `assets/js/main.js`
- Replace `YOUR_LINKEDIN` and `YOUR_REPO` links in `index.html`

## Image files (optional)
Put images in `assets/img/` and name them:
- profile.jpg
- school.jpg
- award.jpg
- smart-aqua.jpg
- ic-tester.jpg
- recipe.jpg
- vba-tools.jpg
- osim.jpg
- maccha.jpg

## Deploy
Repo Settings → Pages → Deploy from branch → `main` → `/(root)`
Make sure `index.html` is in the repository root.
